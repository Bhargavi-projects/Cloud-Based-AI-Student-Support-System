 Hello, today we will understand virtualization in cloud computing. Virtualization is a technology that allows multiple virtual missions to run ony's single physical server. If physical server has hardware resources such as CPU, memory and storage, normally one server runs one operating system. But with virtualization the server is divided into multiple virtual missions. Each virtual mission behaves like an independent computer with its own operating system and applications. This help improve resource utilization reduces hardware cost and increases flexibility. Virtualization is widely used in cloud platforms because it allows efficient sharing of resources and better scalability.
